## Common Questions

### About The Technology

**Q: Is this real AI?**  
A: Yes! We use the same type of AI as ChatGPT, but specialized for images and design.

**Q: Does someone look at my photos?**  
A: No, everything is done by AI automatically. Your photos are private.

**Q: How accurate is it?**  
A: Very accurate for object detection (90-95%). Design suggestions are AI-generated inspiration.

**Q: Can I trust the results?**  
A: The generated designs are artistic interpretations - use them as inspiration, not exact plans.

### About Using It

**Q: How long does it take?**  
A: 30-60 seconds for the full workflow.

**Q: Can I process multiple rooms?**  
A: Yes! Do one at a time, or use the batch feature (coming soon).

**Q: Do I need to download anything?**  
A: No, it runs in your web browser.

**Q: Can I use this on my phone?**  
A: Currently optimized for computers. Mobile app coming in 2026!

**Q: What if I don't like the result?**  
A: Try again with different descriptions, or try just parts of the workflow.
